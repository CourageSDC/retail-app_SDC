## How to start
Install dependencies
```
npm run install
```
Create a production build
```
npm run build
```
Start the server
```
npm run server
```
## Components
This project has 4 main sections for a product each with various functionalities.
1. Product Overview
2. Related Products
3. Questions & Answers
4. Reviews & Ratings
## Technologies
* **API**: Atelier API
* **Back-end**: Node.js, Express.js, webpack
* **Front-end**: React, JavaScript, HTML
